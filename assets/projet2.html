Site Web de Restauration

Un site moderne pour faciliter les réservations et consulter le menu

Voir sur GitHub Démonstration

Ce projet est un **site web de restauration** permettant aux clients de consulter le menu, réserver une table et obtenir toutes les informations sur le restaurant. Le site est responsive et offre une navigation fluide.

Développé en **6 heures**, il intègre des fonctionnalités essentielles pour une gestion efficace des réservations et une présentation attrayante des plats disponibles.

Technologies utilisées

HTML CSS JavaScript PHP MySQL

Caractéristiques principales

  • Page d'accueil attrayante
  • Système de réservation en ligne
  • Consultation du menu
  • Formulaire de contact
  • Informations sur le restaurant
