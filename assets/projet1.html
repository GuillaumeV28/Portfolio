Système de Réservation en Ligne

Ce projet est un **système de réservation en ligne sécurisé**, permettant aux utilisateurs de créer un compte, réserver un créneau et gérer leurs réservations. Il intègre des mesures de sécurité avancées.

**Langages et technologies utilisées** : PHP, MySQL, JavaScript, HTML, CSS **Fonctionnalités principales** : - 🔑 **Authentification sécurisée** avec hashage des mots de passe - 🔄 **Protection CSRF** avec tokens pour éviter les attaques - 📅 **Réservation et gestion des créneaux** - 🛠️ **Interface administrateur** pour gérer les réservations

**Pages principales du site** : - 🏠 `index.php` → Accueil - 👤 `register.php` → Inscription - 🔓 `login.php` → Connexion - 📊 `dashboard.php` → Tableau de bord utilisateur - 🔧 `admin.php` → Gestion des réservations - 📅 `reservation.php` → Réservation d’un créneau
