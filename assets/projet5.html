Fast API

Une application de gestion de tâches full‑stack avec React et FastAPI

Voir sur GitHub Démonstration

Ce projet est une application de gestion de tâches permettant aux utilisateurs de créer plusieurs listes, ajouter, supprimer et marquer comme terminées des tâches, et de consulter un résumé clair de leurs listes.

Construit autour d’un backend performant en FastAPI et d’une interface réactive en React, il assure une expérience utilisateur fluide et une persistance fiable des données grâce à MongoDB.

Technologies utilisées

React JavaScript HTML CSS Python FastAPI MongoDB Docker Nginx Docker Compose

Caractéristiques principales

  • Création et suppression de listes de tâches
  • Ajout, suppression et modification de l’état des tâches
  • Architecture full‑stack avec FastAPI (backend) et React (frontend)
  • Persistance des données avec MongoDB via Motor AsyncIO
  • Déploiement Dockerisé avec Docker Compose et reverse‑proxy Nginx
